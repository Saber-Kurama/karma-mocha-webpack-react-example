# karma-mocha-webpack-react-example

 这个项目主要是用来练习单元测试的,以及karma,webpack,react是如何集成的

# 安装

```
npm install
```

# 测试

```
npm run test         # 进行单元测试并生产代码覆盖率报告

npm run test:watch   # 进行单元测试,并开启监听
```
